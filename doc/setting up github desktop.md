## How to set-up your clone of the game with github desktop 

First you need Ren'py downloaded and installed.
https://www.renpy.org/latest.html

Then you need to find/set your projects directory, Ren'py scans this directory for your current projects.
![10.jpg](https://cdn.steemitimages.com/DQmPssDs3y9iPrQHYo4PoVFmpHV7GcQX7LUwYhS65fYuARo/10.jpg)
![11.png](https://cdn.steemitimages.com/DQmbGsfTomp14hYfySgdjpTbUyRTRDJ88RhMCB6nz4KkmAH/11.png)

Navigate to that directory and create a new folder for this game.
![12.jpg](https://cdn.steemitimages.com/DQmeWe1Y9TZyGUAvDwfYxQq7LY2eYV4SA5xRMNB6EpyZpr1/12.jpg)

Now download and install the GitHub Desktop from:  
https://desktop.github.com/

After opening the installed GitHub Desktop and logging in, go to File>Clone repository...
![01.jpg](https://cdn.steemitimages.com/DQmeVxFgW5pXE5SP8sX5tWNNhxqP2RzimwWDAreWj5nchcQ/01.jpg)

Then:  
1- go to URL.
2- navigate to your renpy projects folder or copy the address here.  
3- paste the games repository `Kiaazad/PrincessTrainer-Jasmine` into the feild and hit clone.
![02.jpg](https://cdn.steemitimages.com/DQmd76x98t7BLc7WfrvXbtywQxaWz55NFrhQiCbMBsZ9irw/02.jpg)

Wait till the game is synced.

Then go back to Ren'py launcher and hit refresh.
![13.png](https://cdn.steemitimages.com/DQmeDoBf41rpjUqxoPwe1ikbQ7yLrPt61cDt6qVef6HBWt5/13.png)

The game will appear on the list, you can choose and run it now.
![14.png](https://cdn.steemitimages.com/DQmUBNnw3eWd69kLyk9WDvTKERRh3aGMraqzwnpNGZBWCoy/14.png)

And you can modify the files to push to the repository, contact me on discord: Kia#6810 if you have any problems.
